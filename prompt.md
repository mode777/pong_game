# Pong Game

- Write a simple pong game thats running in the browser
- The game logic is calculated in 2D space and uses SAT.js for collision detection/solving
- The game is presented in 3D using WebGL and should use Three.js for rendering
- The game features a HUD which displays a simple counter
- Game logic and rendering should be implemented in separate js modules and be loosely coupled. 
- Game logic does not know about rendering
- The game is written in modern javascript (ES6+)
- The game uses CDNs to import the libraries
- Controls are W and S for player 1 and Up and Down for player 2
